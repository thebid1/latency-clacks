## Overview

Snoopers Tenfold is a comprehensive paper trading platform that allows users to test strategies without real capital, featuring realistic market simulation, detailed analytics, and real-time portfolio tracking.

## The Problem

New traders lose money learning. Even experienced traders need a sandbox to test strategies. Existing paper trading platforms are either too simplistic or tied to specific brokers.

## Key Features

- **Real-time Prices**: Live market data feeds
- **Portfolio Tracking**: P&L, drawdown, Sharpe ratio
- **Trade History**: Complete audit trail
- **Strategy Testing**: Backtest and validate approaches

## Live Stats

- **Active Users**: 5 traders
- **Simulated Volume**: 200+ SOL
- **Updates**: Real-time

## Architecture

Full-stack application with Next.js frontend, tRPC API layer, and PostgreSQL database for persistence.
